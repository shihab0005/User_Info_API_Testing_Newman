## Prerequisite:

- Jdk
- Node Js
- Newman
- Html Report Library

## Newman and Report Installation Process:

- Newman Install Command:
```bash
   npm install -g newman
```
- Newman Html Report Install Command:
```bash
  npm install -g newman-reporter-htmlextra
```
